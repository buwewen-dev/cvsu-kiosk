Schema Reference

Top-level: An object with ground and second keys, each a floor.

Floor fields: name, label, outline (SVG path), hallways (array of {x, y, w, h}), rooms (array).

Room fields: id (unique), name, shortName (optional), label (optional override), type, x, y, w, h, isKiosk (boolean), isHighlight (boolean), offices (array).

Room types: admin, academic, lab, library, support, service, public, gate.

Coordinate system: 1210 wide, 620 tall (matches the actual CvSU Cavite City Main Building at 60.5m x 31m, scaled 1m = 20px).
